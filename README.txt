Key mappings for DES396_SpaceWrecked
===============

MENUS
-----
WASD/Arrow Keys - Navigate
Spacebar - Select option
Escape - Return to previous menu

HUD
-----
Note: Press anywhere on the screen when the HUD is displayed to enable the following shortcut keys.
Q - Swap equipment
N - Show navigational arrows
Z - Simulate damage
P - Pause
Spacebar - Speed up credits

*Mouse is not supported.
